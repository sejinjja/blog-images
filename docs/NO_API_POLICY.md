# No API / No Server / No Automation Policy

## Mandatory Rules
- Do not use external APIs of any kind.
- Do not use vision APIs.
- Do not use embedding APIs.
- Do not run local or remote servers for this repository workflow.
- Do not rely on automation scripts as an operation requirement.

## Allowed Method
- Documentation-driven and manual operation only.
- Existing static files in this repository only.

## Scope
- Reuse search for existing images
- Tagging and metadata maintenance
- Review documentation updates

## Out of Scope
- New image generation
- API-assisted image interpretation
